## Project Introduction
This is the first open-source project of our lab. It implements an end-to-end old photo restoration pipeline based on CycleGAN, supporting scratch removal, color restoration, and super-resolution enhancement.

## Project Features
- Lightweight model that runs efficiently on CPU
- Supports batch processing, compatible with Windows / Linux systems
